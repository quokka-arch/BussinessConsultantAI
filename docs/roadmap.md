# Roadmap

## Phase 1: MVP (Weeks 1–6)
- Pick initial niche (recommended: early-stage SaaS)
- Define one use case: idea validation + next-step planning
- Collect 50–100 case studies
- Build intake form + report generation
- Test with first 10 founders
- Improve with human review loop

## Phase 2: Better Advisory Quality
- Add stronger retrieval of similar companies and failure patterns
- Add follow-up chat tied to initial report
- Add company memory and decision tracking
- Improve recommendation explainability

## Phase 3: Operator Assistant
- Weekly KPI check-ins
- Focus prioritization guidance
- Pricing and GTM iteration support
- Runway and risk alerts

## Phase 4: Strategic Advisor
- Scenario planning
- Strategic memo generation
- Multi-quarter planning support
- Team collaboration and board reporting features
